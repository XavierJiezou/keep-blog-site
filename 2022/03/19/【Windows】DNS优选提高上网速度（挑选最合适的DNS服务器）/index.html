笔者在之前的文章详解DNS服务、DNS解析、DNS劫持和污染中已经详细介绍过 DNS 了，今天给大家带来一款免费的 DNS 优选工具（仅适用 Windows），帮助大家提高上网速度，拒绝 DNS 劫持，获得更佳的上网体验。

下载

  • 官网地址：点击下载

  • 备用地址：点击下载

用法

  1. 下载软件后，鼠标右击以管理员身份运行

  1. 如果插的网线，网络连接就选以太网；如果连接的 WIFI，网络连接就选 WLAN。然后点击更新 DNS 列表，更新完成后点击 DNS 测试

  1. 确定更换系统 DNS
