引言

  • <remote>：远程仓库的别名

当然，还有更好的，更加自动化的做法，就是采用 GitHub Actions。首先需要配置 GitHub Actions，提交代码或文件到 GitHub 仓库后，Actions 就会自动将您的代码或文件同步推送到 Gitee 仓库中，具体的配置教程会在下期讲解。
